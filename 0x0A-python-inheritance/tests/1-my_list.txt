Test cases for the MyList class.

Initializing an instance and checking if it's a list:
    >>> my_list = MyList()
    >>> isinstance(my_list, list)
    True

Appending integers and printing the original list:
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Printing the sorted list without modifying the original:
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]
